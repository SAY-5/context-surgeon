# Git workflow (duplicates CLAUDE.md)

Branch names: `feature/<ticket>-<slug>`, `fix/<ticket>-<slug>`,
`chore/<slug>`.

Commits follow Conventional Commits. Rebase to bring branches up to
date; squash to land.

This rule file repeats what is in CLAUDE.md. Ideally it would not exist
— but it does, and it is loaded every session.
